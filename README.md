# 📱 Social Media App

A basic social media UI with posts and interactions.

## 🚀 Features
- Create posts
- Like/comment UI
- Responsive design

## 🛠️ Technologies Used
- React
- JavaScript (ES6+)
- CSS

## 📦 Installation

1. Clone the repository
2. Install dependencies:
```bash
   npm install
```

3. Start development server:
```bash
   npm run dev
```

## 🌐 Usage
After running, open:
http://localhost:5173
